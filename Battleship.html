Game: Battleship

  • Terminal Game
  • Python

Overview:

I designed this game as part of my Introduction to Computer Science coursework, my first ever project. The implemenation included practicing the basic concepts of arrays and loops. This modified battleship game expects a user to find a 4 cell long battleship in a grid of 10 x 10 cells.

Game Design:

As part of implementation a nested array is created to form 2D matrix. Each time user starts the game, a random cell is chosen from the 2D array matrix to be the first cell location and the remaining three cells are randomly decided for their direction and upon the availabilty of cells in that direction. Remaining cells are void.

The user gets 25 attempts and this can be increased or decreased to append difficulty level and allow customization. User is expected to enter a column character (A-J) followed by the row number (0-9). Error checks are implemented in place to check invalid and repeated inputs. A successful guess places an X in the cell, while incorrect guess places a #.
